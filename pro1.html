📖 프로그램 소개

객체 지향의 개념을 누구보다 쉽게 객체 지향 프로그래밍에 대해 깊이 이해하고 적용할 수 있도록 돕기 위해, 블록 코딩을 활용한 시각적 설명을 준비했습니다. 이를 통해 추상화, 캡슐화, 상속, 다형성 등의 개념을 친근하게 전달하며, 복잡한 개념도 쉽게 이해하고 프로그래밍 실력을 향상할 수 있도록 설계했습니다.

🎯 페이지 소개

  • 이론페이지: 다향성, 추상화, 캡슐화에 대한 이론을 애니메이션을 통해서 설명합니다.
  • 문제페이지: 총 3단계의 블록 코딩 문제가 준비되어 있으며, 점점 난이도가 높아집니다.
  • 퀴즈페이지: 마지막으로 이론을 잘 학습했는지 스스로 점검할 수 있도록 퀴즈를 제공합니다.

🙋‍♂️ 역할

- 문제 페이지 및 워크 스페이스(블록 끌어와 쓰는 공간)구현

- 퀴즈 페이지 구현

- Blockly 오픈 소스를 사용하여 블록 코딩 전체 부분 구현
