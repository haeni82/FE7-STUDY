⚙️ HRCOPILOT

📖 프로그램 소개

💡 질문 중심 Copilot + Autonomous DB 질의 자동 연결로 모두를 HR 전문가처럼 만드는 시스템입니다.

Oracle Cloud Infrastructure(OCI)와 LLM 기반 데이터 해석, 검색 증강 생성(RAG) 기술, 자연어 질문 처리 및 데이터 검색 기술, Python 데이터 분석을 활용하여 클라우드 환경에서 산업 및 학술 데이터 분석과 기업 데이터 해석 시스템

🎯 페이지 소개

🙋‍♂️ 역할

  • 사용자에게 채팅으로 질문을 입력받고 Autonomous DB + Gen AI로 분석된 결과를 반환하는 웹페이지 제작
    • 전체 레이아웃 설계
    • 채팅창 구현
      • 봇과 유저 메시지를 구분하여 구현
    • 자동완성 기능 구현
      • 사용자가 띄어쓰기를 입력하면, 백엔드에서 추천 문구를 받아 회색 글씨로 출력
      • 사용자가 탭(Tab)을 입력하면 자동완성 문장이 검정 글씨로 전환
      • 사용자가 엔터(Enter)를 입력하면 질문이 백엔드로 전송됨
    • 가이드라인 기능 구현
    • 파일 업로드 기능
      • 여러 개의 파일 업로드 가능
      • 업로드 진행률을 보여주는 프로그래스바와 퍼센트 표시
      • 파일별 휴지통(삭제) 기능
    • 사이드바 구현
      • 파일 업로드 기능
      • 업로드한 데이터 목록 확인
      • 폴더 및 채팅 추가/삭제 기능
  • 데이터 셋 제작
  • Autonomous DB 테스트
